הוראות ליצירת טבלאות לטבלאות משותפות ב-Access Database:

1. פתח את קובץ ה-Database: App_Data/calnder.db1.accdb.mdb

2. צור את הטבלאות הבאות:

==========================================
טבלה: SharedCalendars
==========================================
שדות:
- Id (AutoNumber, Primary Key)
- Name (Text, 255)
- Description (Memo)
- CreatedBy (Number, Long Integer)
- CreatedDate (Date/Time)

==========================================
טבלה: SharedCalendarMembers
==========================================
שדות:
- Id (AutoNumber, Primary Key)
- CalendarId (Number, Long Integer)
- UserId (Number, Long Integer)
- Role (Text, 50) - "admin" או "member"
- JoinedDate (Date/Time)

==========================================
טבלה: JoinRequests
==========================================
שדות:
- Id (AutoNumber, Primary Key)
- CalendarId (Number, Long Integer)
- UserId (Number, Long Integer)
- Status (Text, 50) - "pending", "approved", "rejected"
- RequestDate (Date/Time)
- Message (Memo)

==========================================
טבלה: SharedCalendarEvents
==========================================
שדות:
- Id (AutoNumber, Primary Key)
- CalendarId (Number, Long Integer)
- Title (Text, 255)
- Date (Date/Time)
- Time (Text, 50)
- Notes (Memo)
- CreatedBy (Number, Long Integer)
- CreatedDate (Date/Time)

==========================================
הערות:
- ודא שיש קשרים (Relationships) בין הטבלאות:
  * SharedCalendars.Id -> SharedCalendarMembers.CalendarId
  * SharedCalendars.Id -> JoinRequests.CalendarId
  * SharedCalendars.Id -> SharedCalendarEvents.CalendarId
  * Users.id -> SharedCalendars.CreatedBy
  * Users.id -> SharedCalendarMembers.UserId
  * Users.id -> JoinRequests.UserId
  * Users.id -> SharedCalendarEvents.CreatedBy
